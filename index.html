whoami

My name is René Tanczos, based in Vienna, Austria. I have a passion about technology, especially coding. You may have seen it written around on this page. I started at school with Visual Basic, C# and the .NET Framework. I was the only student who write apps for windows on a Mac. My first job was as a .NET developer for almost three years. But I always liked art and design. So for me as developer, I wanted to write code for the web. Learning CSS was tough. It is powerful (today even more with animations, transitions and media queries) but sometimes a pain in the ass. Because of all the browser compatibility. Even JavaScript is a bit different to traditional languages, with its async workflow. But now it's one of my favorite languages. Today I try to learn as much as I can. My favorite tool to reach that goal are books. I also build electric circuits and write programmes for microcontrollers. While I'm doing all this things, I try to integrate it seamlessly in our physical world. To create something good, it has to be so simple, that everyone can use it without any explanations.

py-term
Python module to style terminal output, moving and positioning the cursor inside the terminal.
Install through pip. 
pip install py-term
Or get it from source. 
git clone https://github.com/gravmatt/py-term.git
cd py-term
python setup.py install
Usage 
import term

term.writeLine('Hello, Python!', term.green)

Instagram-Profile
Get profiles and media by hashtags

At the time, it is possible to get profile information with its last media entries and media by hashtags.

To get profile informations. 

https://instagram.iive.io/profile/<username>

And to get media entries by hashtag. 

https://instagram.iive.io/hashtag/<hashtag>
By calling this url you will get the latest entries and top posts associated with this hashtag.

To get more informations, visit the project page.

This service is free of use
